# Quick Installation & Run Guide\n\n## 🚀 Fastest Way to Setup (Choose Your OS)\n\n### **For Windows Users** 🪟\n\n1. **Open PowerShell or Command Prompt** in your desired folder\n\n2. **Run the installation script:**\n   ```powershell\n   git clone https://github.com/shindeshailesh962/barcode.git\n   cd barcode\n   install.bat\n   ```\n\n3. **Wait for installation to complete**\n   - Follow any prompts (enter MySQL password when asked)\n   - The script will:\n     - Check prerequisites (Node.js, MySQL, Git)\n     - Clone the repository\n     - Install backend dependencies\n     - Install frontend dependencies\n     - Create the database\n\n4. **After installation, start the application:**\n   ```\n   start.bat\n   ```\n   This will open 2 command windows:\n   - Backend server (port 5000)\n   - Frontend application (port 3000)\n\n---\n\n### **For Mac/Linux Users** 🍎🐧\n\n1. **Open Terminal** in your desired folder\n\n2. **Run the installation script:**\n   ```bash\n   git clone https://github.com/shindeshailesh962/barcode.git\n   cd barcode\n   chmod +x install.sh\n   ./install.sh\n   ```\n\n3. **Wait for installation to complete**\n   - Follow any prompts (enter MySQL password when asked)\n   - The script will:\n     - Check prerequisites (Node.js, MySQL, Git)\n     - Clone the repository\n     - Install backend dependencies\n     - Install frontend dependencies\n     - Create the database\n\n4. **After installation, start the application:**\n   \n   **Terminal 1 - Backend:**\n   ```bash\n   cd backend\n   npm start\n   ```\n   \n   **Terminal 2 - Frontend:**\n   ```bash\n   cd frontend\n   npm start\n   ```\n\n---\n\n## ✅ Verify Installation\n\nWhen everything is running, you should see:\n\n```\n✅ Backend Server:\n   http://localhost:5000\n   API Health Check: http://localhost:5000/api/health\n\n✅ Frontend Application:\n   http://localhost:3000\n   (Browser will auto-open)\n```\n\n---\n\n## 🛠️ Prerequisites Check\n\nBefore running the scripts, ensure you have:\n\n- ✅ **Node.js** v14+ ([Download](https://nodejs.org/))\n- ✅ **MySQL** 8.0+ ([Download](https://www.mysql.com/downloads/))\n- ✅ **Git** ([Download](https://git-scm.com/))\n\n**Check versions:**\n```bash\nnode --version\nmysql --version\ngit --version\n```\n\n---\n\n## 📋 Manual Setup (If Scripts Don't Work)\n\nIf the automated scripts fail, follow the manual setup:\n\n### **1. Clone Repository**\n```bash\ngit clone https://github.com/shindeshailesh962/barcode.git\ncd barcode\n```\n\n### **2. Setup Backend**\n```bash\ncd backend\nnpm install\ncp .env.example .env\n# Edit .env with your database credentials\n```\n\n### **3. Setup Frontend**\n```bash\ncd ../frontend\nnpm install\ncp .env.example .env\n```\n\n### **4. Create Database**\n```bash\nmysql -u root -p\nCREATE DATABASE barcode_system;\nUSE barcode_system;\nSOURCE ../database/schema.sql;\nEXIT;\n```\n\n### **5. Start Backend**\n```bash\ncd ../backend\nnpm start\n```\n\n### **6. Start Frontend**\n```bash\ncd ../frontend\nnpm start\n```\n\n---\n\n## 🔧 Configuration Files\n\nAfter installation, configure these files:\n\n### **backend/.env**\n```env\nPORT=5000\nDB_HOST=localhost\nDB_PORT=3306\nDB_USER=root\nDB_PASSWORD=your_password\nDB_NAME=barcode_system\nJWT_SECRET=your_jwt_secret_key\nCORS_ORIGIN=http://localhost:3000\n```\n\n### **frontend/.env**\n```env\nREACT_APP_API_URL=http://localhost:5000/api\nREACT_APP_ENV=development\n```\n\n---\n\n## 🐛 Troubleshooting\n\n### **Port Already in Use**\n```bash\n# Windows\nnetstat -ano | findstr :5000\ntaskkill /PID <PID> /F\n\n# Mac/Linux\nlsof -i :5000\nkill -9 <PID>\n```\n\n### **MySQL Connection Error**\n- Verify MySQL is running\n- Check database credentials in `.env`\n- Ensure database `barcode_system` exists\n\n### **npm install fails**\n```bash\nrm -rf node_modules package-lock.json\nnpm cache clean --force\nnpm install\n```\n\n---\n\n## 📚 Next Steps\n\nOnce running:\n\n1. **Access Frontend:** http://localhost:3000\n2. **Test API:** http://localhost:5000/api/health\n3. **Read Docs:** \n   - docs/API.md\n   - docs/SETUP.md\n   - docs/ARCHITECTURE.md\n\n---\n\n**Happy Coding! 🎉**\n"
