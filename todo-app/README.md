# Todo List Application with Local Storage\n\nA simple, elegant todo list application with local storage functionality. Built with HTML, CSS, and JavaScript.\n\n## 🎯 Features\n\n- ✅ Add, edit, delete todos\n- ✅ Mark todos as complete/incomplete\n- ✅ Local storage persistence\n- ✅ Filter todos (All, Active, Completed)\n- ✅ Clear all completed todos\n- ✅ Responsive design\n- ✅ No external dependencies\n- ✅ Keyboard shortcuts\n- ✅ Todo categories\n- ✅ Priority levels\n\n## 🚀 Getting Started\n\n### Method 1: Open in Browser\n\n1. Clone the repository:\n   ```bash\n   git clone https://github.com/shindeshailesh962/todo-app.git\n   cd todo-app\n   ```\n\n2. Open `index.html` in your browser\n\n### Method 2: Using Local Server\n\n```bash\n# Using Python 3\npython -m http.server 8000\n\n# Using Python 2\npython -m SimpleHTTPServer 8000\n\n# Using Node.js\nnpx http-server\n```\n\nThen open http://localhost:8000\n\n## 📁 Project Structure\n\n```\ntodo-app/\n├── index.html           # Main HTML file\n├── css/\n│   └── styles.css       # Styles and themes\n├── js/\n│   ├── app.js          # Main application logic\n│   └── storage.js      # Local storage management\n├── README.md           # Documentation\n└── assets/\n    └── icon.png        # App icon\n```\n\n## 💾 Local Storage Structure\n\n```javascript\n{\n  \"todos\": [\n    {\n      \"id\": 1,\n      \"text\": \"Learn JavaScript\",\n      \"completed\": false,\n      \"category\": \"Learning\",\n      \"priority\": \"high\",\n      \"dueDate\": \"2024-12-31\",\n      \"createdAt\": \"2024-06-01T10:00:00Z\"\n    }\n  ],\n  \"settings\": {\n    \"theme\": \"light\",\n    \"filterMode\": \"all\"\n  }\n}\n```\n\n## 🎮 Usage\n\n### Add a Todo\n1. Type your todo in the input field\n2. Press Enter or click the Add button\n3. Todo is automatically saved to local storage\n\n### Edit a Todo\n1. Click the edit icon (pencil)\n2. Modify the text\n3. Press Enter to save\n\n### Delete a Todo\n1. Click the delete icon (trash bin)\n2. Confirm deletion\n\n### Mark as Complete\n1. Click the checkbox\n2.
